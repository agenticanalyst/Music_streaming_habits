An interactive Tableau dashboard that analyzes music streaming data to uncover listener behavior, artist popularity, genre trends, platform performance, and user engagement.

## 🎯 Dashboard Overview

This Tableau dashboard provides interactive visualizations to analyze streaming trends and user listening patterns.

The dashboard includes:

- KPI Cards
- Top 5 Countries by Listeners
- Top 10 Artists by Listeners
- Mood Distribution
- Top Genres by Listeners
- Average Listening Minutes by Platform
- Subscription Distribution
- Offline Mode Usage
- Skip Rate by Genre

## 📌 Key Insights

- Spotify shows the highest user engagement.
- Pop and Hip-Hop are among the most popular genres.
- Premium users spend more time listening.
- User engagement differs across countries.
- Skip rates vary by genre.
- Offline mode is widely used by listeners.
