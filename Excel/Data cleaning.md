# 🎵 Music Streaming Habits Analysis (Excel)

## 📌 Overview
This project analyzes music streaming habits using **Microsoft Excel**. The dataset was explored to identify user listening behavior, subscription trends, genre preferences, and platform usage through interactive visualizations and dashboards.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

## 📊 Dashboard Includes
- Total Users
- Average Listening Minutes
- Platform-wise Users
- Subscription Analysis
- Genre Distribution
- Country-wise Analysis

## 💡 Key Insights
- Premium users spend more time listening than Free users.
- Pop is the most popular genre.
- Mobile is the most used platform.
- User engagement varies across countries.
